# 🛡️ Hitarth Mankodia - Cybersecurity Portfolio

A modern, interactive portfolio website showcasing my journey in cybersecurity, built with React and featuring a stunning Matrix-inspired design.

![Portfolio Preview](https://img.shields.io/badge/Status-Live-success?style=for-the-badge)
![React](https://img.shields.io/badge/React-19.2-blue?style=for-the-badge&logo=react)
![Framer Motion](https://img.shields.io/badge/Framer_Motion-12.35-purple?style=for-the-badge)

## ✨ Features

- 🌐 **Multi-language Greeting** - Welcome animation in 6 languages
- 🎨 **Glassmorphism UI** - Modern glass-effect design with neon accents
- 🎭 **Matrix Rain Effect** - Animated background with falling code
- 🎯 **Interactive Particles** - Mouse-responsive particle system
- 📊 **Animated Skills** - Visual skill meters with dot indicators
- 🏆 **Certificate Gallery** - Embedded PDF viewer for certifications
- 📱 **PWA Ready** - Install as mobile app
- 🌓 **Theme Toggle** - Dark/Light mode switcher
- 📈 **Scroll Progress** - Visual reading progress indicator
- 💼 **Available Badge** - Animated work availability indicator
- 🔗 **Social Bar** - Floating social media links

## 🚀 Tech Stack

- **Frontend:** React 19, React Router
- **Animations:** Framer Motion
- **Styling:** Pure CSS with CSS Variables
- **Build Tool:** Vite
- **Deployment:** Vercel / GitHub Pages

## 🛠️ Installation & Setup

1. **Clone the repository**
```bash
git clone https://github.com/HKrish29/portfolio.git
cd portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Run development server**
```bash
npm run dev
```

4. **Build for production**
```bash
npm run build
```

## 🌐 Deployment

See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed deployment instructions.

**Quick Deploy to Vercel:**
```bash
npm install -g vercel
vercel
```

## 📱 Mobile Installation

Once deployed, users can install the portfolio as a Progressive Web App (PWA) on their mobile devices.

## 🎨 Customization

### Colors
Edit CSS variables in `src/index.css`:
```css
:root {
  --neon-green: #00ff88;
  --neon-cyan: #00d4ff;
  --bg-primary: #06060f;
}
```

### Content
Update personal information in respective component files.
